# Benson.O GitHub Pages Starter

This folder is a minimal GitHub Pages homepage starter for:

```text
https://superobk.github.io/
```

## Use

Create a public GitHub repository named:

```text
superobk.github.io
```

Copy this folder's files into that repository:

```text
_config.yml
index.md
README.md
assets/
```

Then commit and push:

```bash
git add _config.yml index.md README.md assets/
git commit -m "Create personal GitHub Pages homepage"
git push origin main
```

GitHub Pages should publish the site at `https://superobk.github.io/`.

## Maintenance

- Keep `index.md` short and public-facing.
- Update the `Now` section monthly.
- Add project links only after they are public and stable.
- Keep the full source profile in the Obsidian vault at `present/profile/benson-o-profile.md`.
